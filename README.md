# ggf - Windows disk information tool

This one is a "I like `df`. Hey, let's do something similar for Windows." case. Supports Windows XP.

# Examples

- `ggf|clip`

```
DRIVE  TYPE           USED      FREE     TOTAL  TYPE    NAME
C:     Fixed         74 GB    148 GB    223 GB  NTFS    OS
D:     Fixed       1.40 TB    432 GB   1.82 TB  NTFS    DATA

F:     Optical

H:     Fixed        318 GB    146 GB    465 GB  NTFS    OSDisk
I:     Removable     10 GB   3.96 GB     14 GB  FAT32   DD
```
