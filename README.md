# ggf - Windows disk information tool

A lazily-written, df-like, Windows disk information tool.

For base 10 reports, use `-b`.

# Examples

```
D:\>ggf
DRIVE  TYPE           USED      FREE     TOTAL  TYPE    NAME
C:     Fixed       170.81G    52.27G   223.08G  NTFS    OS
D:     Fixed         1.61T   217.42G     1.82T  NTFS    DATA
E:     Fixed       165.35G    67.54G   232.88G  NTFS    VMs
F:     Optical
K:     Fixed         1.48T   345.81G     1.82T  NTFS    BACKUP
```

```
D:\>ggf -P
DRIVE  USAGE
C:     [=============================         ] 76%
D:     [==================================    ] 88%
E:     [===========================           ] 70%
F:
K:     [===============================       ] 81%
```

```
D:\>ggf -M
```
