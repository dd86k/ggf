# ggf - Windows disk information tool

A lazily-written, df-like, Windows disk information tool.

# Examples

- `ggf`

```
DRIVE  TYPE           USED      FREE     TOTAL  TYPE    NAME
C:     Fixed       148.59G    74.49G   223.08G  NTFS    OS
D:     Fixed      1619.22G   243.80G  1863.02G  NTFS    DATA
E:     Fixed       206.36G    26.53G   232.88G  NTFS    VMs
F:     Optical
```

- `ggf -b`

```
DRIVE  TYPE           USED      FREE     TOTAL  TYPE    NAME
C:     Fixed      159.55Gi   79.98Gi  239.53Gi  NTFS    OS
D:     Fixed        1.74Ti  261.77Gi    2.00Ti  NTFS    DATA
E:     Fixed      221.58Gi   28.48Gi  250.06Gi  NTFS    VMs
F:     Optical
```
